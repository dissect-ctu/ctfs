# ctfs

DISsecT@CTU repository for all ctf-related stuff.

## Structure
One folder per CTF, which contains one folder for each task

## Naming conventions
```
<> - compulsory
{} - only in case it is needed
```

### CTF folder
`<YYYY>-<CTF_NAME>_{QUALS|FINALS}`

examples:
`2019-WPICTF`
`2019-WPICTF_QUALS`


### TASK folder
`[<category>]<TASK_NAME> <points>pts` 

examples:
`[pwn]debugme 100pts`
